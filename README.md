# HPC Optimization Project

This project demonstrates data structure optimization in Python by comparing list-based membership lookup with set-based membership lookup.

## Objective
To show how selecting the right data structure improves performance in lookup-heavy workloads.

## Files
- hpc_optimization.py

## Results
- Total data items: 1,000,000
- Total queries: 100,000
- List lookup time: 398.806259 seconds
- Set lookup time: 0.033599 seconds
- Speedup: 11869.62x faster

## How to Run
Run the following command in terminal:

python hpc_optimization.py
